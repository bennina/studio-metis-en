## Quick Start

After purchase, you'll receive a license key. Add it to your project:

```bash
# .env.local
METIS_UI_LICENSE_KEY=your-license-key-here
```

Or configure in `metis.config.js`:

```javascript
module.exports = {
  licenseKey: process.env.METIS_UI_LICENSE_KEY,
};
```
